================================================================================
STEP 6.3: HOLD-OUT TEST SET EVALUATION (留出测试集评估)
================================================================================

STAGE: Predictability Validation
GOAL: 在完全独立的测试集上验证模型的真实预测能力

PCS CONSIDERATIONS:
- Predictability: 获得模型真实泛化能力的无偏估计（核心）
- Computability: 确保测试过程的高效执行
- Stability: 评估测试结果的稳定性和可靠性

================================================================================
BEHAVIORS IN THIS STEP:
================================================================================
1. behavior_1_test_set_construction.txt
   - 构建代表性的独立测试集

2. behavior_2_unbiased_performance_evaluation.txt
   - 执行无偏的性能评估

3. behavior_3_fine_grained_performance_analysis.txt
   - 执行细粒度的性能分析

4. behavior_4_performance_comparison_and_validation.txt
   - 比较和验证测试性能

================================================================================
OVERALL OUTPUT ARTIFACTS:
================================================================================
- test_set_description: 测试集构建说明和统计信息
- test_performance_results: 测试集性能评估结果
- subgroup_analysis_report: 子群体性能分析报告
- temporal_analysis_report: 时间维度性能分析报告
- error_analysis_report: 错误模式分析报告
- performance_comparison: 与基线和CV的性能对比
- comprehensive_test_report: 综合测试评估报告
- confidence_interval_estimates: 性能指标置信区间估计

================================================================================
BEHAVIOR 3.4.3: TIME SERIES ANALYSIS (时间序列分析)
================================================================================

GOAL: 分析时间序列数据的特征和模式（如适用）

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及时间序列建模、趋势分解
  - Model-Agent 擅长应用时间序列分析方法

================================================================================
ACTIONS:
================================================================================

ACTION 3.4.3.1: Identify and Quantify Trends
  趋势分析：长期趋势的识别和量化
  - 使用移动平均识别趋势
  - 拟合趋势线量化趋势
  - 分解时间序列提取趋势成分

ACTION 3.4.3.2: Discover and Model Seasonality
  季节性分析：周期性模式的发现和建模
  - 识别季节性周期
  - 分析季节性强度
  - 建立季节性调整模型

ACTION 3.4.3.3: Detect Anomalous Events
  异常点检测：时间序列中的异常事件识别
  - 使用统计方法检测异常点
  - 识别结构性断点
  - 分析异常事件的特征

ACTION 3.4.3.4: Analyze Autocorrelation
  自相关分析：滞后相关性和周期性验证
  - 绘制ACF和PACF图
  - 识别显著的自相关
  - 确定周期性和滞后关系

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- time_series_analysis_report: 时间序列分析报告（如适用）
- trend_seasonality_decomposition: 趋势季节性分解图

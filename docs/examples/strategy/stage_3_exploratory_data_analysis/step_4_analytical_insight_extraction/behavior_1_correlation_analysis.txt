================================================================================
BEHAVIOR 3.4.1: CORRELATION ANALYSIS (相关性分析)
================================================================================

GOAL: 系统分析变量间的相关关系

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及相关性计算、关系分析
  - Explore-Agent 擅长识别变量间的相关模式

================================================================================
ACTIONS:
================================================================================

ACTION 3.4.1.1: Conduct Linear Correlation Analysis
  Pearson相关系数分析
  - 计算连续变量间的Pearson相关系数
  - 检验相关系数的显著性
  - 识别强相关、中等相关和弱相关关系

ACTION 3.4.1.2: Conduct Nonlinear Correlation Analysis
  Spearman秩相关、Kendall τ相关
  - 计算Spearman秩相关系数处理非线性关系
  - 计算Kendall τ相关处理有序数据
  - 比较不同相关度量的结果

ACTION 3.4.1.3: Analyze Categorical Variable Association
  卡方检验、Cramér's V
  - 进行卡方独立性检验
  - 计算Cramér's V评估关联强度
  - 分析列联表展示分类关联

ACTION 3.4.1.4: Create Correlation Visualizations
  相关性热力图、散点图矩阵
  - 绘制相关性热力图展示整体关联
  - 创建散点图矩阵展示两两关系
  - 生成网络图展示关系网络

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- correlation_analysis_report: 相关性分析报告
- correlation_heatmap: 相关性热力图
- scatterplot_matrix: 散点图矩阵

================================================================================
STEP 3.4: ANALYTICAL INSIGHT EXTRACTION (分析性洞察提取)
================================================================================

STAGE: Exploratory Data Analysis
GOAL: 执行深入的数据分析，提取有价值的洞察和发现

PCS CONSIDERATIONS:
- Predictability: 识别对预测有价值的特征和关系
- Computability: 使用高效的分析算法和方法
- Stability: 验证发现在不同数据子集上的一致性

================================================================================
BEHAVIORS IN THIS STEP:
================================================================================
1. behavior_1_correlation_analysis.txt
   - 系统分析变量间的相关关系

2. behavior_2_grouping_analysis.txt
   - 进行深入的分组对比分析

3. behavior_3_time_series_analysis.txt
   - 分析时间序列数据的特征和模式（如适用）

4. behavior_4_multivariate_statistical_analysis.txt
   - 执行多元统计分析发现复杂模式

5. behavior_5_advanced_visualization.txt
   - 创建高级可视化展示复杂关系

================================================================================
OVERALL OUTPUT ARTIFACTS:
================================================================================
- correlation_analysis_report: 相关性分析报告
- correlation_heatmap: 相关性热力图
- scatterplot_matrix: 散点图矩阵
- grouping_analysis_results: 分组分析结果
- statistical_test_summary: 统计检验摘要
- time_series_analysis_report: 时间序列分析报告（如适用）
- trend_seasonality_decomposition: 趋势季节性分解图
- pca_analysis_results: PCA分析结果
- cluster_analysis_report: 聚类分析报告
- factor_analysis_output: 因子分析输出
- advanced_visualization_gallery: 高级可视化图集
- interactive_dashboard: 交互式分析仪表板

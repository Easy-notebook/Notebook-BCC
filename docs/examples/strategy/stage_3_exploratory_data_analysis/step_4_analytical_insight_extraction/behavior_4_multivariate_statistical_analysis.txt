================================================================================
BEHAVIOR 3.4.4: MULTIVARIATE STATISTICAL ANALYSIS (多元统计分析)
================================================================================

GOAL: 执行多元统计分析发现复杂模式

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及多元统计建模、降维分析
  - Model-Agent 擅长应用高级统计和机器学习方法

================================================================================
ACTIONS:
================================================================================

ACTION 3.4.4.1: Conduct Principal Component Analysis
  主成分分析(PCA)：降维和主要特征识别
  - 执行PCA降维分析
  - 解释主成分的含义
  - 确定保留的主成分数量

ACTION 3.4.4.2: Perform Cluster Analysis
  聚类分析：数据点的自然分组发现
  - 执行K-means聚类
  - 进行层次聚类分析
  - 使用DBSCAN识别密度聚类
  - 确定最优聚类数

ACTION 3.4.4.3: Conduct Discriminant Analysis
  判别分析：组间差异的统计量化
  - 执行线性判别分析（LDA）
  - 评估判别能力
  - 识别判别变量

ACTION 3.4.4.4: Perform Factor Analysis
  因子分析：潜在因子和结构发现
  - 执行探索性因子分析
  - 解释因子结构
  - 计算因子得分

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- pca_analysis_results: PCA分析结果
- cluster_analysis_report: 聚类分析报告
- factor_analysis_output: 因子分析输出

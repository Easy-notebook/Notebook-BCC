================================================================================
BEHAVIOR 3.4.2: GROUPING ANALYSIS (分组分析)
================================================================================

GOAL: 进行深入的分组对比分析

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及分组统计、差异检验
  - Explore-Agent 擅长进行多维度的分组对比分析

================================================================================
ACTIONS:
================================================================================

ACTION 3.4.2.1: Conduct Category-Based Grouping
  按类别分组：不同类别的统计特征对比
  - 按分类变量分组计算统计量
  - 生成分组摘要统计表
  - 可视化组间差异

ACTION 3.4.2.2: Conduct Value-Based Grouping
  按数值分组：基于连续变量的分段分析
  - 对连续变量进行分箱处理
  - 分析各分箱的统计特征
  - 比较不同数值区间的特点

ACTION 3.4.2.3: Conduct Multidimensional Grouping
  多维分组：交叉分组分析和多层级分析
  - 进行多维度交叉分组
  - 执行层级分组分析
  - 分析交互效应

ACTION 3.4.2.4: Perform Statistical Tests
  组间差异检验：t检验、方差分析、非参数检验
  - 进行t检验比较两组差异
  - 进行ANOVA分析多组差异
  - 使用非参数检验（Mann-Whitney, Kruskal-Wallis）

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- grouping_analysis_results: 分组分析结果
- statistical_test_summary: 统计检验摘要

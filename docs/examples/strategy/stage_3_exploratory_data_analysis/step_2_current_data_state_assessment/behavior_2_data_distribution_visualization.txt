================================================================================
BEHAVIOR 3.2.2: DATA DISTRIBUTION VISUALIZATION (数据分布可视化)
================================================================================

GOAL: 通过可视化展示数据分布特征

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及数据可视化、图表生成
  - Explore-Agent 擅长创建多样化的探索性可视化

================================================================================
ACTIONS:
================================================================================

ACTION 3.2.2.1: Create Univariate Visualizations
  直方图、密度图、箱线图
  - 绘制直方图展示频次分布
  - 绘制密度图展示平滑分布
  - 绘制箱线图展示分位数和异常值
  - 绘制小提琴图结合分布和密度

ACTION 3.2.2.2: Compare Distribution Across Groups
  不同组别间的分布比较
  - 按分类变量分组比较分布
  - 绘制分组箱线图和密度图
  - 创建分面图展示多组分布
  - 进行视觉分布差异评估

ACTION 3.2.2.3: Visualize Outliers
  识别和展示异常值模式
  - 使用箱线图标识异常值
  - 绘制散点图展示极端值
  - 创建异常值分布图
  - 可视化异常值的时空模式

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- univariate_visualization_gallery: 单变量可视化图集
- distribution_comparison_charts: 分布对比图表
- outlier_visualization_report: 异常值可视化报告

================================================================================
BEHAVIOR 3.2.1: BASIC STATISTICAL ANALYSIS (基础统计分析)
================================================================================

GOAL: 计算和分析基础统计指标

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及描述性统计、分布分析
  - Explore-Agent 擅长进行统计探索和特征识别

================================================================================
ACTIONS:
================================================================================

ACTION 3.2.1.1: Calculate Descriptive Statistics
  计算均值、中位数、标准差、分位数
  - 计算集中趋势：均值、中位数、众数
  - 计算离散程度：标准差、方差、极差、四分位距
  - 计算分位数：四分位数、十分位数、百分位数
  - 生成综合描述统计表

ACTION 3.2.1.2: Analyze Distribution Characteristics
  评估数据分布的形状、偏度、峰度
  - 计算偏度（skewness）评估分布对称性
  - 计算峰度（kurtosis）评估分布尾部特征
  - 识别分布形状（正态、偏态、多峰）
  - 评估分布的集中和离散特征

ACTION 3.2.1.3: Confirm Variable Type Distributions
  确认连续、离散、分类变量的分布特征
  - 分析连续变量的分布形态
  - 统计离散变量的频次分布
  - 分析分类变量的类别分布
  - 评估变量类型的正确性

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- descriptive_statistics_summary: 描述性统计摘要
- distribution_characteristics_report: 分布特征分析报告

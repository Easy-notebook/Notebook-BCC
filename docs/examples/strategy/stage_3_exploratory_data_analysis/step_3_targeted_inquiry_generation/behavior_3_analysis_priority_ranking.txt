================================================================================
BEHAVIOR 3.3.3: ANALYSIS PRIORITY RANKING (分析优先级排序)
================================================================================

GOAL: 对探索性问题进行优先级排序

RECOMMENDED AGENT: PCS-Agent
  - 该 behavior 涉及优先级评估、资源规划
  - PCS-Agent 擅长基于业务价值进行优先级排序

================================================================================
ACTIONS:
================================================================================

ACTION 3.3.3.1: Rank by Business Importance
  根据业务重要性排序分析问题
  - 评估问题对业务决策的影响
  - 考虑问题的紧急性和重要性
  - 确定优先级得分

ACTION 3.3.3.2: Consider Complexity and Resources
  考虑分析复杂度和资源需求
  - 评估问题的分析难度
  - 估算所需计算资源
  - 评估所需时间和人力

ACTION 3.3.3.3: Develop Execution Timeline
  制定分析执行的时间计划
  - 创建分析任务清单
  - 安排分析执行顺序
  - 设定里程碑和截止日期

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- analysis_priority_matrix: 分析优先级矩阵
- eda_execution_timeline: EDA执行时间表

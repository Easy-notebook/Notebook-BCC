================================================================================
BEHAVIOR 3.3.2: EXPLORATORY QUESTION DESIGN (探索性问题设计)
================================================================================

GOAL: 设计系统化的探索性问题

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及探索性问题设计、分析规划
  - Explore-Agent 擅长设计多维度的数据探索问题

================================================================================
ACTIONS:
================================================================================

ACTION 3.3.2.1: Design Relationship Exploration Questions
  关系探索：变量间的相关性和关联性
  - 识别需要探索的变量对
  - 设计相关性分析问题
  - 规划因果关系探索

ACTION 3.3.2.2: Design Pattern Discovery Questions
  模式发现：数据中的隐藏模式和结构
  - 设计聚类和分组探索问题
  - 规划模式识别任务
  - 确定潜在结构的发现方法

ACTION 3.3.2.3: Design Difference Analysis Questions
  差异分析：不同群体或时期的差异特征
  - 设计组间差异比较问题
  - 规划时间序列变化分析
  - 确定差异显著性检验方法

ACTION 3.3.2.4: Design Trend Identification Questions
  趋势识别：时间序列中的趋势和季节性
  - 设计趋势分析问题
  - 规划季节性模式探索
  - 确定周期性检测方法

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- exploratory_questions_catalog: 探索性问题目录
- relationship_exploration_plan: 关系探索计划
- pattern_discovery_framework: 模式发现框架

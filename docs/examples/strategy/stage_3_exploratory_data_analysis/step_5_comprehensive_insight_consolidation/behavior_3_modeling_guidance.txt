================================================================================
BEHAVIOR 3.5.3: MODELING GUIDANCE (建模指导)
================================================================================

GOAL: 为后续建模提供指导建议

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及特征选择、建模方法推荐
  - Model-Agent 擅长基于EDA结果提供建模建议

================================================================================
ACTIONS:
================================================================================

ACTION 3.5.3.1: Rank Feature Importance
  特征重要性排序：基于EDA结果排序特征重要性
  - 基于相关性排序特征
  - 考虑业务重要性调整排序
  - 识别冗余特征
  - 推荐核心特征集

ACTION 3.5.3.2: Recommend Modeling Methods
  建模方法建议：推荐适合的建模方法和算法
  - 基于数据特征推荐算法类型
  - 考虑问题类型选择方法
  - 评估方法的适用性
  - 提供备选方案

ACTION 3.5.3.3: Identify Preprocessing Requirements
  数据预处理需求：识别额外的预处理需求
  - 识别需要变换的变量
  - 确定编码策略
  - 规划特征工程方向
  - 提出标准化建议

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- feature_importance_ranking: 特征重要性排序
- modeling_recommendations: 建模方法推荐
- preprocessing_requirements: 预处理需求清单

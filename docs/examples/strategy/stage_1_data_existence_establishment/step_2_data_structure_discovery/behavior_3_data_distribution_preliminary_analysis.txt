================================================================================
BEHAVIOR 1.2.3: DATA DISTRIBUTION PRELIMINARY ANALYSIS (数据分布初步分析)
================================================================================

GOAL: 初步分析数据的分布特征

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及统计分析、分布特征探索、异常值检测
  - Explore-Agent 擅长探索性数据分析和可视化

================================================================================
ACTIONS:
================================================================================

ACTION 1.2.3.1: Calculate Basic Statistics
  计算基本统计指标（均值、标准差、分位数）
  - 计算集中趋势指标（均值、中位数、众数）
  - 计算离散程度指标（标准差、方差、极差）
  - 计算分位数（四分位数、百分位数）

ACTION 1.2.3.2: Identify Distribution Characteristics
  识别数据分布特征（正态、偏态、多模态）
  - 分析分布形状（对称性、偏度、峰度）
  - 识别分布类型（正态、指数、幂律等）
  - 发现多峰和异常分布

ACTION 1.2.3.3: Discover Outliers and Extreme Values
  发现异常值和极端值模式
  - 使用箱线图识别异常值
  - 计算Z-score发现极端值
  - 分析异常值的模式和分布

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- statistical_summary: 基本统计摘要报告
- distribution_analysis: 数据分布分析结果
- outlier_detection_report: 异常值检测报告

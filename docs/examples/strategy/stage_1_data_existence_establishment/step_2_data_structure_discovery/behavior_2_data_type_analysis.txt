================================================================================
BEHAVIOR 1.2.2: DATA TYPE ANALYSIS (数据类型分析)
================================================================================

GOAL: 确定和评估每个变量的数据类型

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及数据类型探索、异常检测、类型评估
  - Explore-Agent 擅长数据特征分析和问题发现

================================================================================
ACTIONS:
================================================================================

ACTION 1.2.2.1: Determine Variable Data Types
  确定每个变量的数据类型（数值、分类、文本、时间等）
  - 识别数值型变量（连续、离散）
  - 识别分类型变量（名义、有序）
  - 识别时间型、文本型等特殊类型

ACTION 1.2.2.2: Identify Mixed Types and Anomalies
  识别混合数据类型和异常格式
  - 发现同一列中的混合类型
  - 识别格式不一致的数据
  - 检测类型推断错误

ACTION 1.2.2.3: Assess Type Conversion Necessity
  评估数据类型转换的必要性
  - 确定需要转换的字段
  - 评估转换的可行性和风险
  - 制定类型转换策略

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- data_type_mapping: 变量数据类型映射表
- mixed_type_report: 混合类型和异常格式报告
- type_conversion_plan: 类型转换计划

================================================================================
BEHAVIOR 1.2.1: DATA PATTERN RECOGNITION (数据模式识别)
================================================================================

GOAL: 分析数据的层次结构和关系

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及探索性数据分析、结构发现、关系识别
  - Explore-Agent 擅长数据探索和模式发现

================================================================================
ACTIONS:
================================================================================

ACTION 1.2.1.1: Analyze Hierarchical Structure
  分析数据的层次结构和关系
  - 识别数据的组织层次
  - 分析表间和字段间的关系
  - 绘制数据结构关系图

ACTION 1.2.1.2: Identify Keys and Indexes
  识别主键、外键和索引字段
  - 确定唯一标识字段（主键）
  - 识别关联字段（外键）
  - 发现索引和分区键

ACTION 1.2.1.3: Discover Inter-Table Connections
  发现数据表之间的连接关系
  - 分析表间连接方式
  - 识别一对一、一对多、多对多关系
  - 验证引用完整性

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- data_structure_diagram: 数据结构关系图
- key_index_mapping: 主键外键索引映射表
- table_relationship_report: 表间关系分析报告

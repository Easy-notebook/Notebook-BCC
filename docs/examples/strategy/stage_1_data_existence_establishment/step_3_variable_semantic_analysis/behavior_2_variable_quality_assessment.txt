================================================================================
BEHAVIOR 1.3.2: VARIABLE QUALITY ASSESSMENT (变量质量评估)
================================================================================

GOAL: 评估变量值的合理性和有效性

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及数据质量探索、异常检测、合理性评估
  - Explore-Agent 擅长数据质量分析和问题发现

================================================================================
ACTIONS:
================================================================================

ACTION 1.3.2.1: Assess Value Reasonableness
  评估变量值的合理性和有效性
  - 检查数值变量的取值范围合理性
  - 验证分类变量的类别有效性
  - 评估变量值与业务逻辑的一致性

ACTION 1.3.2.2: Identify Encoding Errors
  识别可能的编码错误或数据录入问题
  - 发现异常编码值
  - 识别数据录入错误模式
  - 检测格式不一致问题

ACTION 1.3.2.3: Verify Value Range Validity
  验证变量取值范围的合理性
  - 确认物理约束（如年龄不能为负）
  - 验证业务规则约束
  - 检查值域边界的合理性

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- quality_assessment_report: 变量质量评估报告
- encoding_error_log: 编码错误和录入问题日志
- value_range_validation: 取值范围验证结果

================================================================================
BEHAVIOR 1.1.1: DATA COLLECTION STRATEGY EXECUTION (数据收集策略执行)
================================================================================

GOAL: 根据项目规划收集所需数据并建立追溯机制

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及执行数据收集代码、文件操作、建立版本控制
  - Define-Agent 擅长执行 Python 代码进行数据操作

================================================================================
ACTIONS:
================================================================================

ACTION 1.1.1.1: Execute Data Collection Plan
  根据Chapter 0的规划收集所需数据
  - 执行预定义的数据收集策略
  - 从指定来源获取数据
  - 记录数据收集的时间和方法

ACTION 1.1.1.2: Verify Data Source Reliability
  验证数据获取渠道的可靠性
  - 检查数据来源的权威性
  - 验证数据传输的完整性
  - 确认数据格式符合预期

ACTION 1.1.1.3: Establish Version Control and Traceability
  建立数据版本控制和追溯机制
  - 创建数据版本标记系统
  - 记录数据变更历史
  - 建立数据血缘追踪机制

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- data_collection_log: 数据收集日志和追溯信息
- data_version_record: 数据版本控制记录
- source_verification_report: 数据来源可靠性验证报告

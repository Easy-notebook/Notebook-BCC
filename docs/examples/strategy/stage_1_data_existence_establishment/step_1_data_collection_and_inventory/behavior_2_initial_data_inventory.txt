================================================================================
BEHAVIOR 1.1.2: INITIAL DATA INVENTORY (初步数据盘点)
================================================================================

GOAL: 统计和验证数据集的基本信息

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及数据统计、文件操作、元数据管理
  - Define-Agent 擅长执行 Python 代码进行数据分析和统计

================================================================================
ACTIONS:
================================================================================

ACTION 1.1.2.1: Collect Basic Data Statistics
  统计数据集的基本信息（行数、列数、文件大小）
  - 计算数据集的行数和列数
  - 记录文件大小和存储格式
  - 统计数据总量和密度

ACTION 1.1.2.2: Verify Data Integrity and Accessibility
  验证数据完整性和可访问性
  - 检查文件完整性（校验和）
  - 验证数据可正常读取
  - 确认所有必需数据都已获取

ACTION 1.1.2.3: Establish Data Catalog and Metadata
  建立数据目录和元数据记录
  - 创建数据资产清单
  - 记录数据来源和描述信息
  - 建立元数据管理系统

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- data_inventory_report: 数据盘点报告（行数、列数、大小）
- metadata_catalog: 数据目录和元数据记录
- integrity_check_results: 数据完整性检查结果

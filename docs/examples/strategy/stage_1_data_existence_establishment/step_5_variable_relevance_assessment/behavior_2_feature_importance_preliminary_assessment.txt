================================================================================
BEHAVIOR 1.5.2: FEATURE IMPORTANCE PRELIMINARY ASSESSMENT (特征重要性初步评估)
================================================================================

GOAL: 使用统计方法评估变量重要性

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及特征重要性评估、统计检验、变量筛选
  - Model-Agent 擅长使用统计和机器学习方法评估特征价值

================================================================================
ACTIONS:
================================================================================

ACTION 1.5.2.1: Statistical Feature Importance Evaluation
  使用统计方法评估变量重要性
  - 使用卡方检验（分类变量）
  - 使用F统计量（数值变量）
  - 使用信息增益或基尼系数

ACTION 1.5.2.2: Identify Redundant and Irrelevant Variables
  识别冗余和不相关的变量
  - 发现与目标变量弱相关的变量
  - 识别信息冗余的变量组
  - 筛选低价值变量

ACTION 1.5.2.3: Establish Variable Priority Ranking
  建立变量优先级排序
  - 综合多种方法的评分
  - 建立变量重要性排名
  - 制定变量筛选策略

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- feature_importance_ranking: 特征重要性排名表
- redundant_variables_list: 冗余变量清单
- variable_selection_strategy: 变量筛选策略文档

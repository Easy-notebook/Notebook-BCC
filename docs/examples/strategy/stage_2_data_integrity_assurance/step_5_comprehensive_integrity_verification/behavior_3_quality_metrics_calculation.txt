================================================================================
BEHAVIOR 2.5.3: QUALITY METRICS CALCULATION (质量度量计算)
================================================================================

GOAL: 计算全面的数据质量度量指标

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及质量指标计算、统计分析
  - Define-Agent 擅长执行 Python 代码进行度量计算

================================================================================
ACTIONS:
================================================================================

ACTION 2.5.3.1: Calculate Completeness Metrics
  计算数据完整性比例
  - 计算字段级完整性比例
  - 计算记录级完整性比例
  - 评估整体数据完整性

ACTION 2.5.3.2: Assess Accuracy Metrics
  评估数据值的准确程度
  - 验证数据值的正确性
  - 评估精度和准确度
  - 计算错误率和准确率

ACTION 2.5.3.3: Measure Consistency Metrics
  测量数据的内部一致性
  - 计算一致性指标
  - 评估规则遵循度
  - 测量数据协调性

ACTION 2.5.3.4: Evaluate Validity Metrics
  评估数据的有效性比例
  - 计算有效值比例
  - 评估格式正确性
  - 测量规范符合度

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- completeness_metrics: 完整性度量结果
- accuracy_assessment: 准确性评估报告
- validity_evaluation: 有效性评估结果
- data_quality_dashboard: 数据质量监控仪表板

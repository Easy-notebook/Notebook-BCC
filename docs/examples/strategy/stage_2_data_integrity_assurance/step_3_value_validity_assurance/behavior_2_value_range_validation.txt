================================================================================
BEHAVIOR 2.3.2: VALUE RANGE VALIDATION (值域验证)
================================================================================

GOAL: 验证数据值域的合理性

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及异常值检测、统计分析
  - Explore-Agent 擅长探索性分析和异常识别

================================================================================
ACTIONS:
================================================================================

ACTION 2.3.2.1: Validate Numerical Range Reasonability
  验证数值变量的取值范围合理性
  - 检查数值是否在合理范围内
  - 识别明显不合理的极端值
  - 验证数值的逻辑约束

ACTION 2.3.2.2: Check Categorical Value Set Completeness
  检查分类变量的取值集合完整性
  - 验证分类值是否在预期集合内
  - 识别无效的分类值
  - 检查分类编码的一致性

ACTION 2.3.2.3: Identify Anomalous Values
  识别异常值和不合理的数据点
  - 使用统计方法识别异常值
  - 分析异常值的分布和特征
  - 区分真实极端值和数据错误

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- value_range_analysis: 值域合理性分析
- anomalous_values_list: 异常值清单

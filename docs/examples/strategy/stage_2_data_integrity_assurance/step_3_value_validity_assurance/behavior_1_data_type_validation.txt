================================================================================
BEHAVIOR 2.3.1: DATA TYPE VALIDATION (数据类型验证)
================================================================================

GOAL: 验证数据类型的正确性和一致性

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及数据类型检查、类型转换
  - Define-Agent 擅长执行 Python 代码进行数据类型验证

================================================================================
ACTIONS:
================================================================================

ACTION 2.3.1.1: Verify Field Data Type Correctness
  验证每个字段的数据类型正确性
  - 检查实际数据类型与预期是否一致
  - 验证数值字段的数值性
  - 确认分类字段的有效性

ACTION 2.3.1.2: Identify Type Inconsistencies
  识别数据类型不一致和转换错误
  - 发现同一字段中的混合类型
  - 识别类型转换失败的记录
  - 检测隐式类型转换问题

ACTION 2.3.1.3: Handle Mixed Types and Format Anomalies
  处理混合数据类型和格式异常
  - 统一混合类型的处理策略
  - 转换或删除格式异常数据
  - 标准化数据类型表示

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- data_type_validation_report: 数据类型验证报告

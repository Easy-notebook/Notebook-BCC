================================================================================
BEHAVIOR 2.2.1: DATA DIMENSION VERIFICATION (数据维度验证)
================================================================================

GOAL: 验证数据集的维度完整性

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及数据结构检查、统计分析
  - Define-Agent 擅长执行 Python 代码进行数据验证

================================================================================
ACTIONS:
================================================================================

ACTION 2.2.1.1: Verify Row and Column Counts
  验证数据集的行数和列数符合预期
  - 检查数据集的实际行数与预期是否一致
  - 验证列数和列名的完整性
  - 比对数据规模与业务需求

ACTION 2.2.1.2: Check Data Structure Integrity
  检查数据结构的完整性和一致性
  - 验证数据表结构的完整性
  - 检查字段定义的一致性
  - 识别结构性缺失或异常

ACTION 2.2.1.3: Identify Dimensional Anomalies
  识别维度异常和结构性问题
  - 发现意外的行或列缺失
  - 识别重复的列或行
  - 检测结构性错误和不一致

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- dimension_verification_report: 维度验证报告

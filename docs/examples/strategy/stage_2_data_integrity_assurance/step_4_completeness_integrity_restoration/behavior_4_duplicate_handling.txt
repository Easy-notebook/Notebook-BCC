================================================================================
BEHAVIOR 2.4.4: DUPLICATE HANDLING (重复记录处理)
================================================================================

GOAL: 识别和处理重复记录

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及重复记录识别、去重操作
  - Define-Agent 擅长执行 Python 代码进行去重处理

================================================================================
ACTIONS:
================================================================================

ACTION 2.4.4.1: Identify Duplicate Records
  识别完全重复和部分重复的记录
  - 检测完全相同的记录
  - 识别关键字段重复的记录
  - 发现模糊匹配的疑似重复

ACTION 2.4.4.2: Analyze Duplication Causes
  分析重复记录的成因和特征
  - 确定重复的来源和原因
  - 分析重复记录的差异
  - 评估重复对分析的影响

ACTION 2.4.4.3: Execute Deduplication Strategy
  制定并执行重复记录的保留或删除策略
  - 制定去重规则和优先级
  - 执行去重操作
  - 记录去重过程和结果

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- duplicate_analysis_report: 重复值分析报告
- deduplication_log: 去重处理日志
- restored_dataset: 完整性恢复后的数据集

================================================================================
BEHAVIOR 2.4.1: MISSING VALUE PATTERN ANALYSIS (缺失值模式分析)
================================================================================

GOAL: 深入分析缺失值的模式和特征

RECOMMENDED AGENT: Explore-Agent
  - 该 behavior 涉及缺失值模式识别、统计分析
  - Explore-Agent 擅长探索性分析和模式发现

================================================================================
ACTIONS:
================================================================================

ACTION 2.4.1.1: Analyze Missing Value Distribution
  分析缺失值的分布模式和规律
  - 统计各变量的缺失率
  - 分析缺失值的空间分布
  - 识别缺失值的时间模式

ACTION 2.4.1.2: Identify Missing Value Types
  识别缺失值的类型（MCAR、MAR、MNAR）
  - 检验完全随机缺失（MCAR）假设
  - 识别随机缺失（MAR）模式
  - 发现非随机缺失（MNAR）情况

ACTION 2.4.1.3: Assess Impact on Analysis
  评估缺失值对分析的潜在影响
  - 评估缺失对统计推断的影响
  - 分析缺失对预测模型的影响
  - 确定缺失处理的优先级

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- missing_value_analysis_report: 缺失值分析报告
- missing_value_type_classification: 缺失值类型分类

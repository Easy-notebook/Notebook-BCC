================================================================================
BEHAVIOR 2.4.2: MISSING VALUE HANDLING STRATEGY (缺失值处理策略)
================================================================================

GOAL: 实施适当的缺失值处理策略

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及缺失值插补、建模方法应用
  - Model-Agent 擅长应用统计和机器学习方法进行插补

================================================================================
ACTIONS:
================================================================================

ACTION 2.4.2.1: Apply Deletion Methods
  对于随机缺失且占比较小的数据应用删除法
  - 列删除：删除缺失率过高的列
  - 行删除：删除含缺失值的行（listwise deletion）
  - 评估删除对样本量的影响

ACTION 2.4.2.2: Apply Simple Imputation
  对数值型和分类变量应用简单填充
  - 均值填充：用均值填充数值型缺失
  - 中位数填充：用中位数填充抗异常值
  - 众数填充：用众数填充分类变量
  - 常数填充：用特定值填充缺失

ACTION 2.4.2.3: Apply Advanced Imputation
  应用高级插值方法处理复杂缺失
  - 回归插值：基于其他变量预测缺失值
  - K近邻插值：使用相似样本填充
  - 多重插值：生成多个可能的插值结果
  - 时间序列插值：利用时间依赖性填充

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- imputation_strategy_document: 插值策略文档
- imputed_data_log: 数据插值日志

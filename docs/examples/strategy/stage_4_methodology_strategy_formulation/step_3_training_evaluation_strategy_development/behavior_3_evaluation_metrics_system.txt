================================================================================
BEHAVIOR 4.3.3: EVALUATION METRICS SYSTEM
================================================================================

GOAL: 建立全面的评估指标体系

RECOMMENDED AGENT: Define-Agent
  - 该 behavior 涉及定义评估指标、建立指标体系
  - Define-Agent 擅长指标定义、体系建立、标准制定

================================================================================
ACTIONS:
================================================================================

ACTION 4.3.3.1: Define Regression Metrics
  回归问题指标定义
  - 准确性指标: MAE(平均绝对误差)、MSE(均方误差)、RMSE(均方根误差)、MAPE(平均绝对百分比误差)
  - 相关性指标: R²(决定系数)、调整R²、Pearson相关系数
  - 鲁棒性指标: 中位数绝对误差、分位数损失、Huber损失

ACTION 4.3.3.2: Define Classification Metrics
  分类问题指标定义
  - 准确性指标: 准确率(Accuracy)、精确率(Precision)、召回率(Recall)、F1分数、F-beta分数
  - 概率评估: AUC-ROC、AUC-PR、对数损失、Brier分数
  - 多类别指标: 宏平均(Macro)、微平均(Micro)、加权平均(Weighted)
  - 混淆矩阵: TP、TN、FP、FN分析

ACTION 4.3.3.3: Define Business-Related Metrics
  业务相关指标定义
  - 成本敏感指标: 加权误差、成本矩阵
  - 时间相关指标: 早期预测准确性、预测提前期
  - 公平性指标: 人口统计平等、机会平等、预测平等
  - 业务KPI: 转化率提升、收入影响、风险降低

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- evaluation_metrics_framework: 评估指标框架
- regression_metrics_definition: 回归指标定义
- classification_metrics_definition: 分类指标定义
- business_metrics_specification: 业务指标规格

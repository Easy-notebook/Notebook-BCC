================================================================================
BEHAVIOR 5.4.1: TRAINING WORKFLOW EXECUTION
================================================================================

GOAL: 执行标准化的模型训练流程

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及准备训练数据、执行批量训练、处理大规模训练
  - Model-Agent 擅长执行模型训练、管理训练流程、监控资源使用

================================================================================
ACTIONS:
================================================================================

ACTION 5.4.1.1: Prepare Training Data
  准备训练数据
  - 执行训练/验证/测试集划分
  - 应用特征预处理管道
  - 验证数据划分的正确性
  - 执行数据泄露检查

ACTION 5.4.1.2: Execute Batch Model Training
  执行批量模型训练
  - 并行训练多个模型
  - 设置训练参数和配置
  - 监控训练进度和资源使用
  - 记录训练时间和计算成本

ACTION 5.4.1.3: Handle Large-Scale Training
  处理大规模数据训练(如需要)
  - 增量训练: 分批训练大数据集
  - 分布式训练: 多机多卡并行训练
  - 内存优化: 数据生成器、梯度累积
  - 检查点保存和恢复

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- trained_models: 训练完成的模型权重
- training_logs: 训练日志和进度记录
- resource_usage_report: 资源使用情况报告

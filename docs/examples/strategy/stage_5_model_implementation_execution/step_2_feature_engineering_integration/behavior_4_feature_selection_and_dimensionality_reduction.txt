================================================================================
BEHAVIOR 5.2.4: FEATURE SELECTION AND DIMENSIONALITY REDUCTION
================================================================================

GOAL: 选择最优特征子集并降低特征维度

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及执行特征选择算法、应用降维技术、验证管道
  - Model-Agent 擅长执行特征选择和降维算法、评估特征重要性

================================================================================
ACTIONS:
================================================================================

ACTION 5.2.4.1: Execute Univariate Feature Selection
  执行单变量特征选择
  - 统计检验: F-test, 卡方检验, 互信息
  - 相关性筛选: Pearson, Spearman相关系数
  - 方差阈值: 过滤低方差特征
  - 单变量排序: 按重要性排序特征

ACTION 5.2.4.2: Execute Multivariate Feature Selection
  执行多变量特征选择
  - 递归特征消除: RFE, RFECV
  - L1正则化: LASSO特征选择
  - 基于模型: 随机森林、XGBoost特征重要性
  - 前向/后向选择: 逐步特征选择

ACTION 5.2.4.3: Apply Dimensionality Reduction
  应用降维技术
  - 线性降维: PCA, 因子分析, ICA
  - 非线性降维: t-SNE, UMAP
  - 特征嵌入: 自编码器降维
  - 验证降维后的性能

ACTION 5.2.4.4: Validate Feature Pipeline
  验证特征处理管道
  - 测试管道的端到端执行
  - 验证特征分布的合理性
  - 检查特征与目标的关系
  - 记录最终特征清单

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- feature_importance_ranking: 特征重要性排序结果
- feature_selection_report: 特征选择过程和结果报告
- final_feature_catalog: 最终特征目录

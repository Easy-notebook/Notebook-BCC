================================================================================
BEHAVIOR 5.2.1: BASIC FEATURE PROCESSING
================================================================================

GOAL: 执行基础的特征处理和转换

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及执行数据类型转换、特征缩放、防止数据泄露
  - Model-Agent 擅长执行特征工程代码、处理数据、验证结果

================================================================================
ACTIONS:
================================================================================

ACTION 5.2.1.1: Execute Data Type Conversion
  执行数据类型转换
  - 类别编码: One-hot, Label, Target encoding
  - 数值处理: 缺失值填充，异常值处理
  - 时间处理: 时间戳解析，时间特征提取
  - 文本处理: 分词，向量化

ACTION 5.2.1.2: Apply Feature Scaling and Standardization
  应用特征缩放和标准化
  - 标准化: StandardScaler, RobustScaler
  - 归一化: MinMaxScaler, MaxAbsScaler
  - 正态化: QuantileTransformer, PowerTransformer
  - 验证缩放后的特征分布

ACTION 5.2.1.3: Prevent Data Leakage
  防止数据泄露
  - 在训练集上拟合转换器
  - 在验证/测试集上应用转换器
  - 验证时间顺序的正确性
  - 检查特征工程的独立性

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- feature_engineering_pipeline: 基础特征处理管道
- transformation_parameters: 特征转换的参数记录
- leakage_prevention_checklist: 数据泄露防止检查清单

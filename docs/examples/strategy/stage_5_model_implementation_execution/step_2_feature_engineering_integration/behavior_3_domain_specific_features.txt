================================================================================
BEHAVIOR 5.2.3: DOMAIN-SPECIFIC FEATURES
================================================================================

GOAL: 构建领域特定的专业特征

RECOMMENDED AGENT: Model-Agent
  - 该 behavior 涉及创建文本特征、时间序列特征、地理空间特征
  - Model-Agent 擅长实现领域特定的特征工程方法

================================================================================
ACTIONS:
================================================================================

ACTION 5.2.3.1: Create Text Features
  创建文本特征(如适用)
  - 基础特征: 长度、词数、字符数
  - TF-IDF特征: 词频-逆文档频率
  - N-gram特征: 单词和字符n-gram
  - 词嵌入特征: Word2Vec, GloVe

ACTION 5.2.3.2: Create Time Series Features
  创建时间序列特征(如适用)
  - 滞后特征: 历史值特征
  - 季节性特征: 季节分解、周期性指标
  - 趋势特征: 移动平均、指数平滑
  - 差分特征: 一阶和高阶差分

ACTION 5.2.3.3: Create Geospatial Features
  创建地理空间特征(如适用)
  - 坐标变换: 投影、距离计算
  - 区域特征: 行政区域、地理编码
  - 空间聚合: 邻域统计、密度特征
  - 空间关系: 距离、方向、邻接关系

================================================================================
OUTPUT ARTIFACTS:
================================================================================
- domain_feature_catalog: 领域特征目录和描述
- domain_feature_code: 领域特征创建代码
